# 打包 imgup 上传工具为独立 exe 文件

## 环境准备

1. 安装 PyInstaller：
```bash
pip install pyinstaller
```

## 打包命令

在项目目录下执行：

```bash
pyinstaller --onefile --windowed --name imgup --icon=NONE upload_gui_standalone.py
```

参数说明：
- `--onefile`: 打包为单个 exe 文件
- `--windowed`: 无控制台窗口（GUI 模式）
- `--name imgup`: exe 文件名为 imgup.exe
- `--icon=NONE`: 不指定图标（可以后续添加）

## 自定义图标（可选）

如果有图标文件 `icon.ico`：

```bash
pyinstaller --onefile --windowed --name imgup --icon=icon.ico upload_gui_standalone.py
```

## 打包后的文件位置

打包完成后，exe 文件位于：
```
dist/imgup.exe
```

## 使用方式

1. 将 `imgup.exe` 复制到任意位置
2. 双击运行
3. 首次启动会自动弹出配置向导
4. 输入图床域名和 Token
5. 配置保存在 `imgup_config.json`（与 exe 同目录）

## 配置文件说明

`imgup_config.json` 示例：

```json
{
  "url": "https://images.example.com/upload",
  "token": "your_secret_token",
  "version": "1.0"
}
```

修改配置后重启程序即可生效。

## 功能特性

✅ 单文件 exe，无需 Python 环境  
✅ 首次启动自动配置向导  
✅ 智能域名识别（自动添加 https）  
✅ 配置文件持久化  
✅ 支持重新配置功能  
✅ 批量上传支持  
✅ 自动复制链接到剪贴板

## 注意事项

1. **杀毒软件警告**：PyInstaller 打包的 exe 可能被误报，需要添加信任
2. **配置文件位置**：exe 文件移动后，配置文件也会重新生成
3. **更新配置**：可以直接编辑 JSON 文件，或使用程序内"重新配置"按钮
