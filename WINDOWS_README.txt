imgup Windows usage
===================

1. Download imgup-windows-x64.exe and double-click it. Python is not required.
2. On first launch, enter your image host domain and upload token.
3. The app creates imgup_config.json next to the exe file.
4. To update, close the app and replace the old exe with the new one.

Notes:
- Put the exe in a writable folder, such as Desktop or Documents.
- Windows SmartScreen may show "Unknown publisher" because the exe is not code-signed.
- You can verify release files with release/SHA256SUMS.txt.
